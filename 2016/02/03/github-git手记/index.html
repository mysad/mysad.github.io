git 的常用命令 MD showTime

git init 初始化

1、git clone url – 克隆

2、git remote add origin url – 增加地址
然后 git pull
本地与gihub的项目关联 无非只有两种情况 一种就是 gihub 有 而本地没有
用clone就行了 第二种就是 github 有 而我本地也有项目 而又想把本地的提交到github这个项目上 那就用第二种方法先关联
clone用在初始化项目
而2的方法把项目与github项目关联上然后就可以推送了

git commit - m “test”

[-m] 这个参数 其实message就是提交日志

git push origin master

origin 就是上面1的origin 应该是url的别名的意思 要不然又要打一串的地址
master 就是主分支

创建分支

创建一个叫做”dev”的分支，并切换过去：
git checkout -b dev

删除分支

要切换另一个分支 才能删除
git checkout master
git branch -d dev

合并分支

如果上面的分支删的话就可以切换主分支
要合并其他分支到你的当前分支（如 master）
git checkout master
git merge dev

推送分支到github去

除非你将分支推送到远端仓库，不然该分支只能你单机玩了别人是不能陪你玩了
git push origin dev
